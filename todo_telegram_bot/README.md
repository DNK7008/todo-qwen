# Telegram Todo Bot для управления задачами

Бот для управления задачами в маленькой компании с тремя уровнями доступа: начальник, руководители и подчиненные.

## 📋 Возможности

### Роли пользователей:
- **👑 Начальник** - полный доступ ко всем функциям
- **👨‍💼 Руководитель** - может создавать задачи и назначать роли
- **👤 Подчиненный** - выполняет назначенные задачи

### Функции:
- ✅ Создание задач с приоритетами
- 📊 Просмотр своих задач
- 📝 Просмотр созданных задач (для руководителей)
- 🔄 Изменение статуса задач
- 👥 Назначение ролей пользователям
- 🔔 Уведомления о новых задачах

## 🚀 Установка и запуск

### 1. Получите токен бота
1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте команду `/newbot`
3. Следуйте инструкциям и получите токен

### 2. Настройка

Откройте файл `bot.py` и замените токен:

```python
BOT_TOKEN = "YOUR_BOT_TOKEN_HERE"  # Вставьте ваш токен
```

### 3. Запуск бота

```bash
cd /workspace/todo_telegram_bot
source venv/bin/activate
python bot.py
```

### 4. Первый запуск

1. Найдите вашего бота в Telegram по имени
2. Нажмите `/start`
3. По умолчанию все пользователи создаются как "Подчиненные"

### 5. Назначение роли начальника

Для назначения роли начальника используйте SQL напрямую или добавьте эту функцию:

```bash
sqlite3 todo_bot.db
UPDATE users SET role = 'chief' WHERE telegram_id = 'ВАШ_TELEGRAM_ID';
```

Чтобы узнать свой Telegram ID, отправьте сообщение боту [@userinfobot](https://t.me/userinfobot)

## 📱 Использование

### Команды:

- `/start` - Запустить бота и показать главное меню
- `/tasks` - Показать мои задачи
- `/create` - Создать новую задачу
- `/my_created` - Показать созданные мной задачи (для руководителей)
- `/set_role` - Назначить роль пользователю (для руководителей и начальника)
- `/help` - Показать справку

### Кнопки быстрого доступа:

При нажатии `/start` появляется меню с кнопками:
- 📋 Мои задачи
- ➕ Создать задачу
- 📝 Созданные мной (только для руководителей)
- ⚙️ Назначить роль (только для руководителей)

## 🛠 Технические детали

- **Фреймворк:** python-telegram-bot v21.0
- **База данных:** SQLite + SQLAlchemy
- **Язык:** Python 3.8+

## 📝 Примечания

- Все пользователи по умолчанию создаются с ролью "Подчиненный"
- Только начальник и руководители могут создавать задачи
- Руководители могут назначать роли другим пользователям
- Задачи имеют 3 уровня приоритета: низкий, средний, высокий
- Статусы задач: Ожидает → В работе → Выполнено → Закрыто
